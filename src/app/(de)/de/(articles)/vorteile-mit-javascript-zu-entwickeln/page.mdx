# Vorteile mit JavaScript zu entwickeln

## Viele unterschiedliche Arten von Anwendungen können damit entwickelt werden

- Webanwendungen
- Moderne Browser unterstützen es direkt.
- [Node.js](https://nodejs.org/) kann für Server-Anwendungen verwendet werden.
- Mobile Anwendungen mit [Cordova](https://cordova.apache.org/).
- Desktop-Anwendungen mit [Electron](https://www.electronjs.org/).

## Eine entwickelte Programmiersprache

- Unterstützung für die meist verbreitesten Programmiersprachen
  - Objektorientierte Programmierung
  - Funktionale Programmierung
- Ein großes Ökosystem.
- Viele Pakete, Werkzeuge und Bibliotheken verfügbar über NPM und GitHub.
- Unterstützt von vielen großen Unternehmen.
- Sehr gute Dokumentation. [MDN](https://developer.mozilla.org/) stellt eine sehr gute Dokumentation bereit.

## Zusammenfassung

Wenn man sich dazu Entscheidet, sich auf eine Programmiersprache zu fokussieren, scheint JavaScript in vielen Fällen eine gute Option zu sein. Für maschinelles Lernen, scheint Python die beste Option zu sein.
